Introduction to JSX

This is an example of JSX, similar to mostly HTML.

JSX is an extension to .js file and is used to:

Structure of React Components → Similar to HTML elements.

Logic + rendering → JSX. Hence better than HTML alone.

Here we can see the typical application like such.

Whatever is written under function App() will be returned when localhost:3000 will be opened.

Just a couple of Observations I made:

  • Unlike NodeJS, here we can directly modify HTML elements itself.
  • And whatever the function App will return will be rendered → But in the centre.
  • Direct HTML snippets are being returned rather than any value or anything.
  • We can’t use for → Hence using map for Array.map()
  • And also we can’t use if() else() → Hence the alternative: direct conditon using ternary operator like:

// Wrong

if(a > 0) {
	return true
}else{
		return false
}

// Right

{
	a > 0 ? "True" : "False"
}
